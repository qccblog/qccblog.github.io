    js中apply和call

    qiucong 2017/3/12 js基础梳理

    最近看到Array.apply(null, { length: 81 })这个方法，由于以前接触的比较少，这次准备花时间好好了解一下。

    # 定义

    # 语法

    fun.apply(thisArg\[, argsArray\])
    1

    # 参数

    thisArg
    在 fun 函数运行时指定的 this 值。需要注意的是，指定的 this 值并不一定是该函数执行时真正的 this 值，如果这个函数处于非严格模式下，则指定为 null 或 undefined 时会自动指向全局对象（浏览器中就是window对象），同时值为原始值（数字，字符串，布尔值）的 this 会指向该原始值的自动包装对象。
    argsArray
    一个数组或者类数组对象，其中的数组元素将作为单独的参数传给 fun 函数。如果该参数的值为null 或 undefined，则表示不需要传入任何参数。从ECMAScript 5 开始可以使用类数组对象。

    # 使用

    # 更改this

    使用apply可以更改fun的this值更改为thisArg。

    function add(a, b) {
    console.log(this);
    return a + b;
}

function add2(a, b){
    return a + b;
}

add(1, 2); //输出Windows对象
add.apply(add2, \[1, 2\]); //输出add2整个函数

    # 实现继承

    通过aplly可以实现继承，因此可以调用父对象的属性和方法，甚至可以以此调用原生对象的方法。

    var Parent = function() {
    this.user = 'li';
    this.show = function() {
        console.log(this.user);
    }
}

var wang = {};
Parent.apply(wang); //wang继承了Parent

function Child() {
    Parent.apply(this); //Child继承自Parent
    //Parent2.apply(this) //很容易就实现了多重继承
    //this.user = 'liu' //将会输出liu
}
var liu = new Child();
liu.show(); //li

    # apply和call

    apply和call唯一的区别就是argsArray这个参数不一样。call方法接受的是一个参数列表，apply接受的数组或者是类数组对象。
    由于这一特性，我们可以简单的给apply使用Array的push等方法传递可变的参数。

    arr = \[1, 2, 3\];
console.log(Math.max.apply(null, arr));
//这个可以传递多个参数，且一次性比较多个值，原生只能一次两个两个比较，类似的例子还有Math.min
//
//如果我们想合并两个数组，直接使用push是不可行的,使用apply可以将arr1当作参数传递给arr
arr1 = \[2, 5, 6\];
Array.prototype.push.apply(arr, arr1);
console.log(arr) //\[1, 2, 3, 2, 5, 6\]

    # Array.apply(null, {length: N})

    {length: N}可以看作一个类数组对象，长度为N，每个元素的默认值就是undefined。
    现在我们回过头去看开始的这个方法，可以看出这个方法是用来创建长度为N的数组，默认值为undefined。我们可以可以给数组指定相关的默认值。

    console.log(Array.apply(null, {length: 3}));
//\[undefined, undefined, undefined\]
console.log(Array.apply(null, Array(3)));
//\[undefined, undefined, undefined\]
//这两种方法的结果是一样的

console.log(Array.apply(null, {1: 'aa', length: 3}));
//\[undefined, "aa", undefined\]
//可以看到我们给数组的第二个元素成功赋值，同样的可以给每个元素单独赋值

console.log(Array.apply(null, {4: 'aa', length: 3}));
//\[undefined, undefined, undefined\]
//如果给不存在元素赋值会不生效
